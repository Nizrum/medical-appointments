# Сервис управления онлайн-записями пациентов
Развёрнутое веб-приложение доступно по ссылке: https://medical-appointments-frontend-nine.vercel.app

## Описание проекта
Веб-приложение для автоматизации записи пациентов на приём в медицинских учреждениях. Система позволяет пациентам просматривать свободные слоты и записываться к врачу, врачам — просматривать своё расписание и отмечать завершение приёма, а администраторам — управлять расписанием, карточками врачей и услугами.

## Основные функции
- **Пациент**: просмотр врачей и свободных слотов, запись на приём с указанием жалоб, отмена записи, просмотр своих записей.
- **Врач**: просмотр расписания и списка записанных пациентов, отметка о завершении приёма.
- **Администратор**: управление врачами и услугами, настройка расписания (шаблоны недели и отдельные слоты), подтверждение/отмена/перенос записей.

## Технологический стек
- **Backend**: FastAPI, SQLAlchemy, Pydantic
- **Database**: PostgreSQL
- **Frontend**: Vue.js 3, Pinia, Vue Router, Tailwind CSS
- **Containerization**: Docker, Docker Compose

## Локальный запуск через Docker Compose
### Предварительные требования
- Установленные Docker и Docker Compose
- Git

### Инструкция
1. Клонировать репозиторий
    ```
    git clone https://github.com/Nizrum/medical-appointments.git
    cd medical-appointments
    ```

2. Создать файл окружения
    ```
    cp .env.example .env
    ```

3. Отредактировать файл .env при необходимости (изменить пароли, порты и т.д.)

4. Запустить приложение
    ```
    docker-compose up
    ```

5. Доступ к приложению
- Frontend (веб-интерфейс): http://localhost
- Backend API (документация Swagger): http://localhost:8000/docs

### Остановка приложения
```
docker-compose down
```

### Остановка с удалением данных БД
```
docker-compose down -v
```
